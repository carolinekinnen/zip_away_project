INSTRUCTIONS FOR RUNNING THE APPLICATION

1.	Open a terminal and navigate to ckinnen-cschippits-shashab/
2.	Run:
    chmod 775 install.sh run_scraping.sh
3.	To run the data cleaning scripts and launch the application, type:
	./install.sh
4.	Open a web browser and navigate to http://127.0.0.1:8000/.
5.	In the zip code field, type in a starting zip code.
6.	Use the drop-down menu to choose a target state.
7.	Check any combination of boxes corresponding to preferences. At least one box must be checked.
8.	Click “Submit”
9.	Wait patiently while the results populate.
10.	Read the results and have fun interacting with the map!
11.	(optional) To run the web scraping scripts, type:
	./run_scraping.sh
